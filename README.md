Online Sales Excel Dashboard


Project Overview
- This project is an interactive Excel dashboard designed to analyze online sales performance across regions, product categories, and customer transaction behavior.

The dataset is structured into three layers:
- Raw Data sheet
- Working sheet (data transformation & calculations)
- Dashboard sheet (final visualization)



Business Objective
- To analyze revenue performance, transaction behavior, product category contribution, and regional sales trends using Excel-based reporting.



Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Transformation



Key Performance Indicators (KPIs)
- Total Revenue
- Number of Transactions
- Preferred Mode of Payment



Dashboard Features

- Region-wise revenue analysis (Asia, Europe, North America)
- Top product categories analysis
- Category-wise revenue breakdown
- Units sold per category
- Interactive filtering using region slicer



Interactive Features
- Region-based slicer (Asia / Europe / North America)
- Dynamic KPI updates
- Interactive charts and visuals
- Filter-driven analysis



Key Insights
- Revenue varies significantly across regions.
- A few product categories contribute most of the revenue.
- Customer transaction behavior differs by region.
- Payment preferences vary across markets.

  - Raw Data sheet

  - Dashboard sheet
